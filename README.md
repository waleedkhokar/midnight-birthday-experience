# 🎂 Birthday Experience - Javascript 🎀

An interactive cinematic birthday website with emotional storytelling. A premium, quiet, and deeply personal birthday experience designed to feel like a short emotional film rather than a typical birthday website.

## 🛠️ Tech Stack

| Technology | Purpose |
|------------|---------|
| **Next.js 13.5** | React framework for SSR & routing |
| **TypeScript** | Type safety & better developer experience |
| **Tailwind CSS** | Utility-first styling |
| **GSAP** | Smooth timeline animations & scene transitions |
| **Framer Motion** | UI micro-interactions & hover effects |

---

## 🎯 Features

- ✅ **8 Cinematic Scenes** - Emotional storytelling journey
- ✅ **Auto-play Background Music** - User-controlled with mute/unmute
- ✅ **Interactive Cake Cutting** - Knife animation with slice separation
- ✅ **Floating Messages** - Typewriter-style emotional wishes
- ✅ **Particle Effects** - Stars, confetti, and floating elements
- ✅ **Glassmorphism UI** - Premium blur effects & gradients
- ✅ **Fully Responsive** - Works on all devices
- ✅ **Cinematic Transitions** - Smooth GSAP-powered scene changes

| Scene | Description |

## 🚀 Run Locally

### Prerequisites

- Node.js 18+ 
- npm or yarn

### Installation

```bash
# Clone the repository
git clone https://github.com/waleedkhokar/midnight-birthday-experience.git


# Navigate to project
cd birthday-experience

# Install dependencies
npm install

# Run development server
npm run dev
```
